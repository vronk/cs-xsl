Stylesheet data2view_tei.xsl

Description

Stylesheet for formatting TEI-elements inside a FCS/SRU-result. the TEI-elements are expected without namespace (!) (just local names) This is not nice, but is currently in results like that.

The templates are sorted by TEI-elements they match. if the same transformation applies to multiple elements, it is extracted into own named-template and called from the matching templates. the named templates are at the bottom.

Template classnames

Description

This template generates the CSS classes from any element following the convention:

  • namespace prefix (tei) + local name
  • if present: namespace prefix + "type" + value of @type
  • if present: namespace prefix + "rend" + value of @rend
  • if present: namespace prefix + "place" + value of @place
  • if present: namespace unit + @unit name + value of @unit

The various parts of a name are concatenated with hyphens. Blanks in @rend are replaced by underscores.
